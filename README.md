# Калькулятор квадратных уравнений

> **Документация подготовлена для промежуточной версии проекта и описывает реализованный на данный момент функционал.**
>
> По мере развития проекта README будет актуализироваться и дополняться новыми возможностями.

## Описание проекта

Консольное приложение на C++, предназначенное для решения квадратных уравнений вида:

ax² + bx + c = 0

Программа поддерживает как классический ввод коэффициентов, так и ввод уравнения в математической форме.

---

## Реализованные возможности

### 1. Решение квадратных уравнений

Пример:

1 -5 4

Результат:

Решение найдено:

D = 9

Количество корней: 2

x1 = 4

x2 = 1

---

### 2. Поддержка линейных уравнений

Пример:

0 2 -4

Результат:

x1 = 2

---

### 3. Обработка особых случаев

Бесконечное количество решений:

0 0 0

Результат:

INF

Нет решений:

0 0 5

Результат:

NO SOLUTION

---

### 4. Проверка корректности ввода

Примеры ошибочного ввода:

abc

1 2

1 2 3 4

Результат:

WRONG

---

### 5. Ввод коэффициентов

Пример:

1 -5 4

---

### 6. Ввод уравнения строкой

Пример:

x^2 - 5x + 4 = 0

После разбора программа определяет коэффициенты:

a = 1

b = -5

c = 4

и выполняет решение уравнения.

Также поддерживаются записи:

2x^2 - 7x + 3 = 0

-x^2 + 5x - 8 = 0

x^2 + x + 1 = 0

3x^2 + 2 = 0

---

## Структура проекта

### Quadrat.cpp

Главный файл программы.

Отвечает за взаимодействие с пользователем, получение ввода и запуск решения.

### Parser.h / Parser.cpp

Разбор и проверка входных данных.

Функции:

* проверка чисел;
* определение типа ввода;
* разбор уравнения вида x^2 - 5x + 4 = 0;
* извлечение коэффициентов a, b и c.

### Solver.h / Solver.cpp

Математический модуль.

Отвечает за вычисление дискриминанта и нахождение корней уравнения.

### EquationResult.h

Структура для хранения результата решения.

Содержит:

* тип результата;
* список корней;
* значение дискриминанта.

### ResultPrinter.h / ResultPrinter.cpp

Вывод результата решения в удобном для пользователя виде.
