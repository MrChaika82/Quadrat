# Quadratic Equation Solver

Консольное приложение на C++ для решения квадратных и линейных уравнений.

## Возможности

* Решение квадратных уравнений
* Решение линейных уравнений
* Комплексные корни
* Проверка корректности ввода
* Unit tests через Google Test

## Используемые технологии

* C++
* Visual Studio
* Google Test

## Пример ввода

1 -5 4

## Пример вывода

OK
4
1
